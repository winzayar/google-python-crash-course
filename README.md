This repository is organized by the course's weekly modules. Each folder contains the corresponding quizzes, practice exercises, and scripts I wrote.

* ### `/Week 1 - Hello Python/`
    * **Topics:** Introduction to programming, Python syntax, functions (`print()`, `type()`), and arithmetic operations.

* ### `/Week 2 - Basic Python Syntax/`
    * **Topics:** Data types, variables, conditional statements (`if`, `else`, `elif`), and defining/calling functions.

* ### `/Week 3 - Loops/`
    * **Topics:** `while` and `for` loops, `range()`, and common loop errors.

* ### `/Week 4 - Strings, Lists, and Dictionaries/`
    * **Topics:** Advanced string manipulation (indexing, slicing, formatting), list methods, and dictionary operations.

* ### `/Week 5 - Object-Oriented Programming/`
    * **Topics:** Defining classes and methods, object instances, inheritance, and importing Python modules.

* ### `/Week 6 - Final Project - Word Cloud/`
    * **Project:** A final script that applies all the concepts from the course. This script reads a text file, counts the frequency of each word, and generates a "word cloud" image.
